##### **AI AFTERLIFE** 

##### 

###### **AI-Powered Memorial Persona Platform**



AI After Life is a full-stack Django web application that creates AI-powered digital memorial personas of loved ones. The platform allows users to preserve the communication style, personality, and voice of a person by training an AI persona on their WhatsApp chat exports and voice recordings.

###### 

###### **Project Overview**



This project addresses the deeply human need to preserve memories and connections, combining cutting-edge AI/ML technologies with an intuitive, emotionally sensitive user experience. Once trained, users can hold real-time text or voice conversations with the persona—hearing responses that authentically reflect their unique way of speaking.



###### 

###### **TECH STACK** 



* **Backend:** Django 4.2, Django REST Framework



* **AI/LLM:** Groq API(LLaMA-based models).



* **Voice Cloning:** Coqui TTS(XTTS v2), Bark.



* **Audio Analysis:** Librosa, PyTorch, pydub.



* **Database:** SQLite(dev)/PostgreSQL(prod).



* **Frontend:** Bootstrap 5, Vanilla JS, Web speech API.
* 



###### 

###### **Key Features** 



* **Persona Creation:** Create named memorial personas with relationship context and personal details for more authentic interactions.



* **WhatsApp Chat Training:** Upload .txt chat exports; the system parses messages, extracts personality traits, communication patterns, common phrases, and emotional tone.



* **Voice Cloning:** The system analyzes pitch, timbre, rhythm, and MFCCs to clone and reproduce speech.



* **Training Progress Tracking:** A 9-step training pipeline with real-time progress updates and accuracy metrics calculated using confusion matrix evaluation.
*
